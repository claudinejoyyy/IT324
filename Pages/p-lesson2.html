Lesson 2 - The Hypertext Transfer Protocol
Hypertext Transfer Protocol
  • An application layer communications protocol used to access resources on the World Wide Web
  • Invented by Tim-Berners Lee
  • It was jointly developed by the World Wide Web Consortium (W3C) and Internet Engineering Task Force (IETF)
Version History
  • HTTP 0.9 - an out-dated version that is only kept for historical purposes
  • HTTP 1.0 - a rarely used version nowadays
  • HTTP 1.1 - the most frequently used today
    • RFC 2616 - the SOMETHING SOMETHING SOMETHING
  • HTTP 2 - an increasingly popular version of HTTP that is being used more and more
    • Comparably lower level
    • Patterned after SPDY (developed by Google which is an open source, Transmission Control Protocol based, application layer that transports content over the Web)
    HTTP Fundamentals
  • HTTP runs on top of TCP/IP using TCP Port 80 by default or TCP Port 443 for HTTPS
  • HTTP is based on a client-server architecture
  • HTTP uses a request response standard protocol
  • HTTP is a stateless communication protocol (serve and forget)
  • HTTP provides support for other functionalities, such as:
    • Client
  • Sometimes called user agents
  • Can be a...
    • Web browser - a software program that allows a user to locate, access, and display web pages
    • Web crawlers - a program or automated script which browses the World Wide Web in a methodical, automated manner
      • A Google bot is an example of a web crawler
    • Web applications - a software program that runs on a web server
    Servers
    • This provides services that the client consumes
    • Some examples of servers include
      • Origin server - actual machine where needed data is stored
      • Proxy server - verifies and forwards incoming client requests to other servers for further communication
      • Gateway - type of security solution that prevents unsecured traffic from entering an internal network of an organization
      • Tunnel - llows for the secure movement of data from one network to another
