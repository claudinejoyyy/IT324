Lesson 1 - Hypertext Markup Language

Hypertext Markup Language (HTML)

  • Basic building block of the web
  • HTML defines the content of a webpage
  • Official language for making a webpage

History

  • 1989 - Tim Berners-Lee invents the Web with HTML as its publishing language
  • July 1994 - HTML specification for HTML 2 is released
  • March 1995 - HTML 3 is published as an Internet Draft
  • January 1997 - HTML 3.2 is ready

Versions of HTML

  • HTML 1
  • HTML 2.0
  • HTML 3.2
  • HTML 4.01
  • XHTML
  • HTML 5
References
