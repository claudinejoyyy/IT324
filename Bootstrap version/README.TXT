The homepage of our course website can be accessed by clicking on index.html
This page will have 4 navigation bars on top indicating the reference for the index, prelims lessons, midterms lessons, and for the tentative final lessons.
All pages for the lessons are in the "pages" folder.
All images used for the course website are in the "img" folder.
All scripts are in the "js" folder.
All styles are in the folder "css" folder.
This website is not currently hosted in any hosting sites, thus can only be accessed locally
