    Java Server Pages - JSP

      What?

    • A server-side programming technology that enables the creation of dynamic, platform-independent method for building Web-based applications.

      • When?

    • first appeared in 1998. The official versions, 1.0 and 1.1, both appeared in 1999, and both were very popular. The current version, 1.2, appeared in 2001, and is the most popular implementation yet.

      • Who?

    • Sun Microsystems

      • Advantages

      Performance
    • since JSP allows embedding Dynamic Elements in HTML pages, it is significantly better when it comes to performance compared to other similar technologies.

      • Segregation
    • designers can design web page separately and servlet programmers can insert the dynamic contents separately.

      • Platform-independence
    • JSP will work with different platforms that are currently available.

      • Lower Learning Curve
    • JSP is easier to learn and apply compared to other similar technologies.
