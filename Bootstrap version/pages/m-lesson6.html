        • Example: font, background
      • Longhand properties

  2. Values
    • keyword - not enclosed in quotes (example: initial)
    • numbers - integers and reals in decimal notation (example: font-weight: 100)
    • dimension - number with a unit (2px, 3em)
      • length units
        • font-relative: em (font size)
        • font-relative: ex (height of lowercase)
        • font-relative: ch (zero character)
        • font-relative: rem (root-element)
        • viewport-percentage: vw (1% of viewport width)
        • viewport-percentage: vh (1% of viewport height)
        • viewport-percentage: vmin
        • viewport-percentage: vmax
        • absolute-length: cm, mm, q, in, pt, pc, px
      • angle units: deg, grad, rad, turn (1 full 360)
      • duration units: s, ms
      • frequency units: hz, khz
      • resolution units: dpi, dem, dppx
      • percentages
      • URLs and URIs - for specifying rsources
      • color
        • keyword based (e.g. red)
        • #fff - hexadecimal digits
        • rgb ( __,__,__ ) {e.g. rgb (255,255,255) }
        • rgbg ( __,__,__ ) -- alphachannel-transparency
        • hsl -- hue, saturation, lightness {e.g. hsl (0, 150%, 50%) }
