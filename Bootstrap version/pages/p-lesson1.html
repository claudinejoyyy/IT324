Lesson 1

The Web VS. The Internet

The Internet

  • A global network of networks that connects computers world wide
  • A global information system
  • An internetwork

History

  • The internet started as a way for government researchers to share information
  • 1962 - "Galactic Network" was proposed by J.C.R. Licklider
    • Galactic Network - Computers that can talk to one another
  • 1969 - Birth of the internet
    • ARPAnet delivered its first message: node-to-node communication from one computer to another. (The first computer was located in a research lab at UCLA and the second was at Stanford; each one was the size of a small house.) The message - LOGIN - was short and simple, but it crashed the fledgling ARPA network anyway: The Stanford computer only received the note's first two letters.
  • 1972 - E-mail was born, and became the first killer technology

Internet of Things

  • Sometimes abbreviated to IoT
  • It refers to the connection of computing devices to the internet

Internet Protocol

  • Sometimes abbreviated to IP
  • "Protocol" is defined as a standard way of doing things

Wired and Wireless Technologies

Wired tech Wireless tech
UTP Infrared
Fiber Bluetooth

The World Wide Web

  • An information system on the Internet that allows documents to be connected to other documents
  • Sir Tim Berners-Lee is recognized as the Father of the Modern Web

History

  • 1989 - Birth of the World Wide Web
  • 1990 - Three fundamental technologies were written by Tim Burner's Lee
    1. HTML : HyperText Markup Language
    2. URI : Uniform Resource Identifier
    3. HTTP : Hypertext Transfer Protocol
  • 1994 - Sir Tim founded the World Wide Web Consortium (W3C)
  • 2009 - Sir Tim established the World Wide Web Foundation

Wide Area Information Servers

  • Sometimes abbreviated to WAIS
  • An index of databases

Gopher

  • A protocol named after an animal known for digging
  • Going deeper into a hierarchy to look for a file
