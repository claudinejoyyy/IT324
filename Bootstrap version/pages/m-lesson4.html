    Lesson 11

    CSS Selectors

    CSS Selectors

    • Used as a condition in the CSS rule
    • Subjects - matched selectors
    • Selector syntax - chain of one or more sequences of simple selectors separated by combinators with one pseudo-element

    Simple selector

    • Type selector - type of elements you are selecting
    • Universal selector
    • Attribute value - elenents are targeted based on a value
      • abbr[title] <- [attr]
      abbr[title = xyz] <- [attr = value]
      p[class = xyz] <- [attr ~= value]
      [lang |= en] <- [attr |= value]
      • for CSS3:
      [attr ^= value]] starts with this
      [attr $= value] end with this
      [attr *= value] found anywhere
    • Class selector - depends on class value
      • p.xyz (xyz is the class)
    • ID selector - for the ID attribute (which should be unique for valid HTML)
      • #xyz
    • Pseudo-class
      • Dynamic pseudo-class
        • link pseudo-classes (for hyperlinks)
          • :link
          • :visited (link was already seen)
        • user action pseudo-class (for user interaction)
          • :hover (when pointer is hovered on something)
          • :active (when you click on something)
          • :focus (interaction with the element)
      • Target pseudo-class (exists in CSS 2.1)
        • target (CSS 3) - allows formatting a section that is referenced by a bookmark 
          • :target {
	color:red;
}
      • Language pseudo-class
        • lang()
          • :lang(en) {

}
      • UI element states pseudo-class (interactive user elements)
        • enabled
        • disabled
        • checked
        • indeterminate
      • Structural pseudo-class
        • root targets root element of the document
        first-child formats ALL element that are the first child of their parents
        last-child formats ALL element that are the last child of their parents
        only-child element with no siblings
        nth-child value should be indicated
        nth-last-child similar to nth-child, only counting from the end
        first-of-type
        last-of-type
        only-of-type
        nth-of-type()
        nth-last-of-type()
        empty - targets empty selectors
        not() negates, reversing the logic

    CSS combinators

    • Descendant combinator (whitespace, i.e., space, tab, line feed, carriage return, form feed)
    • Child combinator (>)
    • Sibling combinatory
      • Adjacent sibling combinator (+) 
        h1+p
        (same level)
      • General sibling combinator (+) 
        h1~p
        (any sibling)

    CSS Pseudo-elements

    • ::first-letter
    • ::first-line
    • ::before
    • ::after
    • Example: 
    • h1::before{
	content:”Topic”;
}
body{
	counter-reset: tn 0;
}
h1{
	counter-increment:tn 1;
}
h1::before{
	content “Topic” counter(tn) “:”;
}

    Document object model tree

    • A model that JavaScript can use to access and change all the elements of an HTML document.
