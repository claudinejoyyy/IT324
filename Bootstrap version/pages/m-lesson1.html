    Lesson 8

    Hypertext Markup Language

    Hypertext Markup Language

    • Often abbreviated as "HTML"
    • The basic building block of the web
    • A programming language that defines the content and stucture of a web page
    • HTML development is overlooked by the World Wide Web Consortium (W3C)
    • The W3C also specify the documents for the HTML standards

    History

    • 1989 - Tim Berners-Lee invents the Web with HTML as its publishing language
    • July 1994 - HTML specification for HTML 2 is released
    • March 1995 - HTML 3 is published as an Internet Draft
    • January 1997 - HTML 3.2 is ready

    Versions of HTML

    • HTML 1.0 - released 1989
    • HTML 2.0 - released 1995
    • HTML 3.0
    • HTML 3.2
    • HTML 4.0 - released 1997
    • HTML 4.1 - released 1999
    • HTML 5 - released 2014
      • Release was triggered by the rise of mobile devices
    • HTML 5.1

    Related file formats

    • XML
    • XHTML

    HTML Syntax

    • Byte Order Mark
      • Bytes that represent the Unicode points
      • Can be ignored
    • Any number of comments or space characters
      • Syntax is <!-- comment -->
    • Root element in the form of <html> element
      • Metadata (head)
      • Body
      • Elements can be empty or void, or have no closing tag
        • e.g. <br>
      • An "alt" attribute is necessary for all images
    • Character entity reference
      • & amp - ampersand
      • & lt - lesser than
      • & gt - greater than

    HTML 5

    • A collection of technologies
    • DOCTYPE declared as <!DOCTYPE html>

    HTML 4.01 DOCTYPE Declarations

    • Strict
      • Deprecated elements can not be used
      • <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
    • Loose or Transitional
      • Deprecated elements can be used
      • <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
    • Frameset
      • Similar to transitional, though a frameset can be used
      • <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">
    • Content model
    • Tag omission
    • Content attributes
    • Global attributes
    • ARIA role attributes
    • DOM interface
    • Root Element
      1. General attributes
      2. Element-specific attributes
      3. Data-* attributes
      4. Role, Aria-* Attributes
        • WAI - Web Access Initiative
