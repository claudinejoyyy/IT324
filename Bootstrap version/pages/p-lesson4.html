HTTP Resource Addressing

HTTP Resource Addressing

https://name:pw@website.com:2000/bank/accounts.txt;sessionid=289?cat=imp&name=dan#history
  • HTTP resources are identified using URIs (specifically, HTTP URLs)
    • Scheme (http)
      • Could either be http: or https:
    • Authentication information (name:pw)
      • Optional
    • Host (website.com)
      • Optional
    • Port number (2000)
      • Optional
    • Path to the resource bank/accounts.txt
      • States the location of the file on the document root of the server
    • Scheme-specific parameters sessionid=289?
      • Optional
    • URL-encoded query cat=imp&name=dan
      • Optional
    • Fragment #history
      • A fragment or a bookmark
      • Generally used to identify headings for easier navigation
