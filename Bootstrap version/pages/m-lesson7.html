Lesson 14

Client-side Scripting with JavaScript

JavaScript

  • HTML and CSS have little interactivity (passive). This is where JavaScript comes in.
  • Associates a program with the web page
  • A program/script is downloaded and executed in the browser

Associating JavaScript

  1. Externally linked (can be placed anywhere)
    • <script type = "text/javascript" src = "script.js">
  2. Embedded within HTML
    • <script>
	console.log('embedded script...')
</script>
  3. In-line script
    • <button onclick = console.log (“inline script…”);>Click me</button>
  • Code defined outside of a function is global or top level.
  • Code defined inside of a function is function body

Attributes

  • Defer attribute
    • Code does not execute until document has been rendered
    • Fetcing does not block the rendering
  • Async attribute
    • Fetch the script, render the page, then execute even the if rendering is not yet done.
    • Some browsers do not support defer and async, in this case, placing will be the key
    • For browsers with no scripting support or with scripting support disabled: <noscript> No Scripting Support. </noscript>
  • Some browsers disable JavaScript for security measures purposes.

Window object

  • Represents a browser environment
  • To manipulate a document:
    • window.document or document
  • To return a reference by element:
    • window.document or document
  • To set an attribute:
    • h.setAttribute(‘title’)
  • To manipulate the styling:
    • h.style = “color: blue”
  • To retrieve all elements of a familiar style
    • [p, p, p, p]
  • querySelector()
    • Argument will be a CSS selector
    • If there is more than one match, it will choose the first
  • querySelectorAll()
    • Will return all matches except pseudo elements
    • document.querySelectorAll(‘body p’)
